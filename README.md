# Abogados Cuenca — Dr. Luis Mora Samaniego

Sitio web institucional del estudio jurídico **Abogados Cuenca** (Cuenca, Ecuador).

## Contenido

Sitio estático (HTML + CSS + JavaScript, sin dependencias) con navegación por pestañas:

| Pestaña | Contenido |
|---|---|
| Inicio | Presentación, diferenciadores y "¿Por qué elegirnos?" |
| Nosotros | Nuestra Historia, Misión y Visión |
| Valores | Los 8 valores corporativos |
| Servicios | Las 8 áreas del derecho con su detalle |
| El Abogado | Experiencia profesional y áreas de experiencia |
| Contáctenos | Datos de contacto y formulario que abre WhatsApp |

## Estructura

```
index.html            Todo el contenido, dividido en pestañas
assets/css/styles.css Identidad visual: negro, dorado y blanco (Georgia + Arial)
assets/js/main.js     Navegación por pestañas, menú móvil y formulario → WhatsApp
Imagen/               Logo, sello y fotografías del despacho
```

## Desarrollo local

Abre `index.html` en el navegador, o levanta un servidor estático:

```bash
npx serve .
```

## Despliegue

Publicado con GitHub Pages desde la rama `main`.
